# Secure Login System

A secure login web application developed using Python Flask.

## Features

- User Registration
- Secure Login System
- Password Hashing using bcrypt
- SQL Injection Protection
- Session Management
- Logout Functionality

## Technologies Used

- Python
- Flask
- SQLite
- bcrypt

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python app.py
```

## Default URL

```text
http://127.0.0.1:5000
```
